## Installation

Add a file name `config.yml` into the same directory as `package.json` using the following template:

```yml
token: discord bot token here
guildId: "guild id here"
channelId: "channel id here"
prefix: hw

color:
    red: 0xff0000
    green: 0x04CF8D
    blue: 0x28BAD4
    yellow: 0xebc934
    aqua: 0x34ebbd
```
